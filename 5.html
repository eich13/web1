Git Hub

깃허브 에 가입해서 New repository를 클릭해 새로운 저장소를 만든 뒤, 작업한 파일을 업로드 한다. 수정사항이 생길 때 마다 계속 새로 업로드를 하면 됨.
저장소에 파일을 모두 업로드 한 뒤 setting 클릭, github Pages 메뉴에서 source를 none 에서 master branch 로 변경하면 된다. 그러면 웹사이트 주소를 갖게 됨.



사진 참고.

HTML파일만을 호스팅 하는 것을 static web hosting이라고 부른다. 이 경우 무료 서비스도 많고, 쉽게 작업이 가능함. 본격적인 개발 언어를 사용할 경우 Dynamic hosting을 사용해야 하고, 무료가 아닌 것도 많다.

현업에서도 호스팅 업체에 맡기는 경우가 많음. https://eich13.github.io/web1/ 나의 웹사이트 링크
